### **四、扩展：使用Python计算离散互信息**
```python
import numpy as np
from scipy.stats import entropy

# 联合概率矩阵 P(X, Y)
P_xy = np.array([[0.1, 0.4], [0.3, 0.2]])

# 边缘分布 P(X) 和 P(Y)
P_x = np.sum(P_xy, axis=1)  # [0.5, 0.5]
P_y = np.sum(P_xy, axis=0)  # [0.4, 0.6]

# 计算熵
H_x = entropy(P_x, base=2)  # 1.0 比特
H_y = entropy(P_y, base=2)  # ~0.971 比特
H_xy = entropy(P_xy.flatten(), base=2)  # 需展平为1D数组，但更准确的方法是手动计算

# 手动计算联合熵
H_xy_manual = -np.sum([p * np.log2(p) for row in P_xy for p in row if p > 0])  # ~1.845 比特

# 互信息
I_xy = H_x + H_y - H_xy_manual  # ~0.126 比特
print(f"互信息 I(X; Y) = {I_xy:.3f} 比特")
```

**输出**：
```
互信息 I(X; Y) = 0.126 比特
```

通过以上步骤和例子，可以系统化地计算任意两个随机变量的互信息。
